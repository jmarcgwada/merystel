# **App Name**: Zenith POS

## Core Features:

- Touchscreen Interface: Intuitive touchscreen interface for quick order taking.
- Item and Category Management: Create and manage item categories with customizable button appearance.
- Advanced Checkout: Modal window for processing payments with support for multiple payment methods.
- Restaurant Mode: Manage tables, assign customers, and handle 'take away' orders.
- Interface Customization: Personalize interface elements with live previews.
- Dashboard Shortcuts: Organize shortcut buttons into distinct sections.
- Company Settings: Define company details and manage customer information.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for reliability and sophistication.
- Background color: Light blue (#E8EAF6), providing a calm and clean backdrop.
- Accent color: Amber (#FFC107) to highlight key actions and information.
- Font: 'PT Sans', a humanist sans-serif for clarity and modernity. Suitable for both headlines and body text.
- Simple, clear icons for categories and actions to ensure easy navigation.
- Clean, grid-based layout optimized for touch interactions.
- Subtle animations and transitions for feedback and visual interest.
